This is the Beta Version of my FlowerClock App. Simply Download the entire file and run the control.exe and press start or stop. It still contains testing features that allow you to kill the program with the esc hot key. The source code is more up to date, but it is all in development.

The clock is read my using the large leaf behind the flower as an hour hand. Petals are even markers, spaces are odd. The bee is the minute hand and each wing flap represents one second.

This program is (so far) only compatible with Windows devices.
